秉持著這股衝勁，在努力自學的過程中找到了老師的影片，發現老師的教學方式淺顯易懂，所以想加入WeHelp Bootcamp 的行列， 我一定會好好把握機會，不讓家人和您失望的！

請描述一件你曾經碰到最困難的事情，如何克服?
在一次主管指派擔任領導者時，要在短時間內指導新人工作內容，並要讓他能融會貫通，是 一件非常不容易的事，因為每個人對於工作的態度及積極度大有不同，自己的期待也有所不同，導致情緒失控、事倍功半。
最後選擇了收斂自己的脾氣並修身養性，把新人當小孩教導，理性處理及不斷地溝通後得到 很棒的反饋及成就感，就如同彭彭老師一樣。

我想盡可能地把老師在 HTML 及 CSS 課程裡教的內容都特別運用在申請網頁上， 例如：問題 框有特別為了 RWD 而設計、作息時間特別做出表格呈現，方便且可迅速瀏覽、在表格內設計 hover 當隱藏功能，有藍白相反背景及依序呈現 彩虹色階的文字變化，以及相對位置的運用，標題背景及段落首行縮兩格部分雖然目前還沒學到，但有另外上網查 CSS 並成功運用上。
在開發過程中，若顯示畫面能如同理想設計一般呈現，而不用偵錯，會有很大的成就感及興奮感。
